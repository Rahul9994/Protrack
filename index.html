FIREBASE SETUP
One-time setup (~5 min) to enable cross-device sync. Free forever on Firebase's Spark plan.
STEP 1 — CREATE PROJECT
Go to console.firebase.google.com → Add project → Name it protrack → Disable Google Analytics → Create project.
STEP 2 — GET CONFIG
Inside your project → Click ⚙️ Project Settings (gear icon) → scroll to "Your apps" → Click the </> Web icon → Register app as protrack-web → Copy the firebaseConfig object shown.
STEP 3 — ENABLE GOOGLE LOGIN
Left sidebar → Authentication → Get started → Google → Enable → Add your email as support email → Save.
STEP 4 — CREATE DATABASE
Left sidebar → Firestore Database → Create database → Choose Start in test mode → Pick any region near India (e.g. asia-south1) → Done.
STEP 5 — AUTHORIZE YOUR DOMAIN
In Authentication → Settings → Authorized domains → Add your GitHub Pages URL (e.g. yourusername.github.io).
STEP 6 — PASTE CONFIG INTO THE FILE
Open protein_tracker.html in Notepad or VS Code. Find the firebaseConfig block near the top. Replace each PASTE_YOUR_... value with your real values. Save the file. Re-upload to GitHub Pages.
